一路记 WayLog 是个人旅行控制台，也是你的旅行手账。
把计划、行程、收藏、预算和备忘收拢到一个地方——不再散落在备忘录、聊天记录和截图里。

下载应用 GitHub
为什么需要它

旅行信息太散了，
把它们装进一个地方

旅行计划在备忘录里，酒店地址在聊天记录里，想去的地方在地图收藏夹里，花费记在脑子里……一路记把它们都收拢起来。

📱

信息散落各处

旅行计划、酒店地址、餐厅收藏分散在备忘录、聊天记录、地图收藏夹和截图里，出发前总要翻半天。

📅

行程排得混乱

每天的安排靠脑子记或者写在笔记本上，改一个时间就要全部重排，临出发了还在纠结先去哪后去哪。

💰

预算只在事后算

旅行中花钱没概念，回来才发现超预算了。缺少一个随手就能记、边花边看的简单工具。

核心功能

覆盖旅行全流程
10 大功能模块

从规划到出发，从记录到回顾，一路记覆盖了旅行中最常用的每一个场景。

✈️

旅行管理

创建、编辑和查看旅行，支持计划中 / 旅途中 / 已完成三种状态管理，支持置顶。

📋

每日行程

按日期组织行程，记录时间、地点、备注和费用，时间排序一目了然。

📍

地点收藏

19 种图标覆盖景点、餐厅、酒店、交通等场景，集成高德 POI 搜索。

💰

预算与记账

设置行程预算，7 种费用分类（交通/住宿/餐饮/门票/购物/活动/其他），支持多币种。

🚗

交通与住宿

航班、火车、自驾、巴士多种交通方式，住宿信息管理（名称、地址、入住/退房时间）。

✅

打包与备忘

可勾选的打包清单 + 支持置顶的旅行备忘，证件信息、航班号、注意事项不再到处翻找。

☁️

云同步

基于 Supabase 的 RLS 安全同步，增量 upsert + 版本冲突检测，多设备数据保持一致。

🔐

多种登录

游客 / 邮箱验证码 / 手机验证码 / 微信四种登录方式，灵活选择，数据安全可控。

🌤️

天气与地图

集成高德地图和 Open-Meteo 天气，在应用内查看位置、天气和路线，规划更直观。
