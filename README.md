# 一路记 WayLog 官网

这是一个用于微信开放平台审核和应用信息展示的极简静态官网项目。

## 文件

- `index.html`：官网首页
- `privacy.html`：隐私政策
- `terms.html`：用户协议
- `styles.css`：页面样式

## 部署方式

这是纯静态网站，不需要构建命令，也不需要后端服务。可以直接部署到 GitHub Pages、Vercel、Netlify、Cloudflare Pages 或任意静态网站托管服务。

部署后可在微信开放平台填写：

- 官网 URL：静态托管服务生成的首页地址
- 隐私政策 URL：静态托管服务生成的 `privacy.html` 页面地址
- 用户协议 URL：静态托管服务生成的 `terms.html` 页面地址
